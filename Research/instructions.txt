=====<how 2 set up source>=====
this should be the same for all sources
create a folder called Trunk2012 in your root folder (C:/Trunk2012)
then create two folders, one named Build and the other name Contribs (you can just put them in Trunk2012 it doesn't matter) (C:/Trunk2012/Build, C:/Trunk2012/Contribs)
export all your contribs into the Contribs folder (boost should look like this: C:/Trunk2012/Contribs/boost_1_56_0)
if you have to compile them, do so.
create a system path variable called CONTRIB_PATH (like this: https://web.archive.org/web/20230511202921if_/https://camo.githubusercontent.com/06d461203787f0048526a056aebd4e21a0d6567a91261d5a7b310e6536417756/68747470733a2f2f696d616765732d6578742d322e646973636f72646170702e6e65742f65787465726e616c2f4b77657a5a33337875485972306c5a58597141346370544d564f57646a66436c6452734a775875504245732f68747470732f692e696d6775722e636f6d2f6c3953784776722e706e67) with the contrib path there (mines is C:/Trunk2012/Contribs)
