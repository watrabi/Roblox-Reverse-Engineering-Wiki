# Mobile Patching — Android — 2018L+

*Credits: DoggoITA (author)*

1. Decompile APK
2. Edit all roblox.com → your domain in smali
3. Edit public key in libroblox.so
4. Compile APK
5. If connection error: `v1/setting/application?applicationName=AndroidApp` (2018 uses Setting/QuietGet/AndroidApp)
6. After fflags setup, landing page with login/signup should appear

## Endpoints

- `v2/login` — different response formats for 2019M- vs 2019L+
- `v2/signup` — same as login
- `alerts/alert-info` — returns alert text in 2019L+
- `v1/join-game` — acts like placelauncher (2020L/M+)
