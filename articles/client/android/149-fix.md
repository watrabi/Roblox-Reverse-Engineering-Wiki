# Fix Error 149 on Mobile (Android/iOS)

*Credits: meditation/meditexts*

After patching the client, if you get error 149 joining RCC:

## Version Check Fix

The version check requires the first 7 version numbers + platform suffix:

- Example: `"2.271.9androidapp"`, `"2.271.9iosapp"`
- Format: `{7versionchars}{platform}app`

If running RCC, reset your gameserver after adding the version.
