# Client Patching — Windows — 2013–2017E

*Credits: meditation/meditext (author)*

## Player (2013–2017E)

Tools: x32dbg, RBX SigTools, proper site

### Domain Replacement

1. Open client in x32dbg → Symbols → RobloxPlayerBeta.exe
2. Click [Az] icon for strings
3. Search "roblox.com" → dump → replace with 10-char domain
4. Open AppSettings → replace BaseURL

### Public Key Replacement

1. Use RBX SigTools → open PublicKeyBlob.txt
2. Search "Bgiaa" in strings
3. Open in dump → replace

## RCCService

Same steps as Player

### Infinite JobId

HxD: search `00 00 00 00 00 c0 82 40` → change to `00 00 f8 1f 5f a0 02 42`

> ⚠️ Not recommended for production — causes memory leakage.
