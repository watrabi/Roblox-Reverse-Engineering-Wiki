# Client Patching — Windows — 2018+ (VMProtected)

*Credits: shikataganai, aep (VMP patch), VisualPlugin (theory), VMsLover (Init error 4 guide)*

## VMP Client Patch

1. Download [ASLR Removal plugin](https://github.com/buzzer-re/x64dbg-ASLR-Removal/releases/tag/1.0)
2. Place in x32dbg/plugins folder
3. Open client in x32dbg → Plugins → ASLR Removal → Remove ASLR
4. Rename output to .exe, open with -help arg
5. Attach x32dbg to client (File → Attach)
6. Memory Map → vmpx → hex editor → find empty space
7. Put URLs: `.example.com` then `example.com`
8. Copy address, search roblox URLs, replace push addresses

## Bypass Initialization Error 4

1. Launch x32dbg without exe
2. Options → Preferences → uncheck all except System Breakpoint
3. Restart with player.exe
4. Click blue arrow 2× → Symbols → select exe
5. Click [Az] to see VM-protected strings
6. Select .vmpx in Memory Map
7. Dump → find empty space → put URLs
8. Copy address, replace roblox URL push instructions
9. Do same for BGIAA

## Patch Tickets (2018+)

Search "ProcessTicket exception" → JE → JMP. Use DFFlagCLI120342 or debuglocalrcc fflag with localhost JMP.
