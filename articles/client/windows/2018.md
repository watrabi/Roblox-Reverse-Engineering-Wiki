# Client Patching — Windows — 2018M

*Credits: Aep (author)*

1. Open RobloxPlayerBeta.exe in x32dbg → Symbols → double-click
2. Right-click CPU → Search For → Current Module → String References
3. Search "Client:Connect" → scroll up → JE above "localhost" → JMP
4. Search "Loading shader files" → JE above → JMP
5. File → Patch File → save
6. Open patched exe in HxD → replace roblox.com with 10-char domain
7. Replace public key (BgIAA, the one with "==" at end)
8. Do same for RCCService.exe
