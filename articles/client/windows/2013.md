# Client Patching — Windows — 2013–2014L

*Credits: cub, ORC guide server community*

## Tools Required

x32dbg, HxD, RBXSigTools, PHP files, core scripts

## Site Patch

1. Open client in x32dbg, Symbols → double-click exe
2. Click [Az] icon, search roblox.com
3. Right-click → Follow In Dump → Ctrl+E to replace
4. Change appsettings.xml

## Blocking %s

Search → double-click → scroll up → JE → JMP

## Analytics / Game

Search analytics → double-click → scroll up → JNE → JMP

## Public Key

Search BGIAA → generate key → PublicKeyBlob.txt → follow in dump → replace

## Invalid Request

Search → JNE/JE → JMP on each

## !isKeyboardAcquired

push 5 → push 6
