# Client Patching — Windows — 2021E

*Credits: Jetray#4509, iknowidontexistbutwhatifwin#7604 (authors)*

## Tools

- x32dbg
- HxD
- stud_pe

## Client Patching

1. In appsettings.txt: `http://www.roblox.com` → `http://localhost/.test`
2. Open client in x32dbg → Symbols → click right arrow 2× (bypasses VMP)
3. Right-click → Search For → Current Module → String References
4. Search "trust check failed" → JNE/JE → JMP
5. Search "127.0.0.1" → find JE above with "push D188" under → JMP
6. Create ClientSettings/ClientAppSettings.json with proper fflags
7. Replace ssl/cacert.pem
8. Download required DLLs, extract to client folder
9. Open stud_pe → drag client → Functions → Add Import → Injector.dll → add func

## Server/RCC Patching

1. HxD: search `00 68 74 74 70 73 00` → `00 68 74 74 70 00 00`
2. x32dbg: patch trust check (no VMP arrow needed)
3. Patch "Non-trusted BaseURL used. HttpRbxApiService is only for Roblox API calls" → JMP
4. Add DevSettingsFile.json and gameserver.json

## Launch

```
Server: RCCService.exe -Console -verbose -placeid:1818 -localtest "gameserver.json" -settingsfile "DevSettingsFile.json" -port 64989
Client: start RobloxPlayerBeta.exe -a "http://localhost/Login/Negotiate.ashx" -j "http://localhost/game/placelauncher.ashx" -t "1"
```
