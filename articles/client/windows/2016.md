# Client Patching — Windows — 2016L

*Credits: ORC guide server community, table (Uranium loading screen)*

## Invalid Request 5

1. Search "invalid request 5"
2. Double-click → go up to JNE/JE → JMP it

## Remote SysStats

1. String search "sysstats"
2. Double-click "Players::onRemoteSysStats disconnect"
3. JMP both JNE and JE → push 1 → push 0
4. Back to References
5. Double-click "Players::onRemoteSysStats disconnect not in the clist"
6. JMP JE above it, then JMP JNE above it
7. push 1 → push 0

## Analytics

1. Search "analytics" → double-click JUST analytics
2. Go up to "game" → JMP the JNE

## Public Key (HxD)

1. Open HxD, search bgiaa
2. Replace with PublicKeyBlob.txt contents

## Domain (HxD)

Replace all roblox.com → 10-char domain. Change appsettings.xml too.

## RCCService Only

HxD: search `00 00 00 00 00 c0 82 40` → `00 00 f8 1f 5f a0 02 42`

## Version Compatibility

Search "versioncompatibility" → replace with 00's in hex

## API Key Patch

Change `api.roblox.com/%s/?apikey=%s` → `www.yourdomain.com/%s/?apikey=%s`

## MD5 Hashes & Security Versions

Create folders: GetAllowedMD5Hashes, GetAllowedSecurityVersions. Return JSON with client hash and version.
