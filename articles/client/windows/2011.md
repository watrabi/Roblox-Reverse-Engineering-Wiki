# Client Patching — Windows — 2011e–2012m

*Credits: valley (author), nukley (name/id method), matt (terrain patch), dxf*

## Player Names and IDs

Works for 2011E–2012M (not RobloxPlayer).

1. Open client in x32dbg
2. View → Modules → select RobloxApp.exe
3. Symbols tab → type `_CxxThrowException` → breakpoint
4. Spam run until window appears
5. Create a new game (File → New)
6. Command bar: `game.Players:CreateLocalPlayer(0):LoadCharacter()`
7. Select player, change name — game will hang
8. CPU tab → bottom-right highlighted area → double click → jmp the je/jne/jnz above

## Terrain Server Hosting Patch (2011L–2012M)

1. Open client in HxD
2. Search hex: `8B 75 DC C6 45 FC 00 3B`
3. Line above should have `E8 .. .. .. 00` — fill with `90`
4. Save

## Optional: Keyboard Windows Key Fix

1. Search `!isKeyboardAcquired`
2. Double-click top result
3. Scroll down to `push 5` → change to `push 6`

## Rocky Patch 101

*Credits: Rocky*

1. String search "Running Script"
2. Below it find JNE → change to JE
3. Search "Failed to start network client on port"
4. Below is localhost → change JE to JMP
