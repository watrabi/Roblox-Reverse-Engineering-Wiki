# thumbnails.roblox.com Web API v1

*Description: REST API for requesting Roblox thumbnails, captured from the Wayback Machine (2020-11-24)*
*Credits: juliaoverflow/roblox-web-apis, Roblox Creator Hub docs, daniel176 (compilation)*

Base URL: `https://thumbnails.roblox.com`

## Endpoints

### `GET /v1/asset-thumbnail-animated`

Fetches an animated thumbnail for an asset.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `assetId` | integer | Yes | The asset id |
| `Roblox-Place-Id` | integer | No | Place ID context |

### `GET /v1/assets`

Fetches thumbnails for assets.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `assetIds` | array | Yes | The asset ids |
| `Roblox-Place-Id` | integer | No | Place ID context |
| `returnPolicy` | string | No | `PlaceHolder`, `ForcePlaceHolder`, `AutoGenerated`, `ForceAutoGenerated` |
| `size` | string | No | e.g. `420x420`, `512x512`, `768x432` |
| `format` | string | No | `Png`, `Jpeg`, `Webp` |
| `isCircular` | boolean | No | Circular crop output |

### `GET /v1/assets-thumbnail-3d`

Fetches 3D thumbnail data for an asset.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `assetId` | integer | Yes | The asset id |
| `useGltf` | boolean | No | Return in glTF format |
| `Roblox-Place-Id` | integer | No | Place ID context |

### `POST /v1/batch`

Returns thumbnails with varying types and sizes in a single request.

Body: `ThumbnailBatchRequest` objects supporting types: Avatar (1), AvatarHeadShot (2), GameIcon (3), BadgeIcon (4), GameThumbnail (5), GamePass (6), Asset (7), BundleThumbnail (8), Outfit (9), GroupIcon (10), DeveloperProduct (11), AvatarBust (12), AutoGeneratedAsset (13), PlaceIcon (14), AutoGeneratedGameIcon (15), ForceAutoGeneratedGameIcon (16), Look (17), CreatorContextAsset (18), Screenshot (19).
