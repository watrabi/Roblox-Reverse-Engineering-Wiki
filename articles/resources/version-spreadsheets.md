# Roblox Version Spreadsheets

*Description: Community-maintained spreadsheets tracking historical Roblox mobile versions with download links*
*Credits: ✝️ Silicon ✝️ (iOS), LuaWiz (Android)*

Community-driven spreadsheets cataloging Roblox mobile version history with release dates, identifiers, and direct download links.

## iOS Versions

- [ROBLOX iOS Versions](https://docs.google.com/spreadsheets/d/1qAN8Eh4iPjO1aECiO1tclTXTLKfCka3StReywLJ3A58/edit) by ✝️ Silicon ✝️
- Tracks versions from 1.0 (May 2011) onward
- Columns: Version, Release Date, Archived?, Software Version, External Identifier, Minimum iOS, Notes, Download Link
