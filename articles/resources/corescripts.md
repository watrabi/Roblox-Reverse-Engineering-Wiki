# CoreScripts

*Description: Archive of Roblox CoreScripts from 2010 to 2019*
*Credits: lrre.wiki (original documentation)*

CoreScripts are essential to make clients operate and function in modern Roblox.

First appeared: 2011. Originally streamed as signed asset files (rbxsig format). In 2014 (v0.167.1.51539), changed to allow local corescripts in content/scripts folder.

## Download

- [CoreScripts 2010–2019 Archive](../../../resources/CoreScripts_2010_-_2019.7z) (919 KB)

## Key Behaviors

- Scripts have RobloxScript security context
- Multiple signing formats (see Signatures)
- Executed in their own Lua virtual machine

Official corescripts: github.com/ROBLOX/Core-Scripts
