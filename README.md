# Roblox Reverse Engineering Wiki

Community-sourced Roblox patching and reverse engineering guides, originally compiled from the ORC guide server community and lrre.wiki archives. Sorted and organized into Markdown articles by AI.

## Structure

```
articles/
├── client/          # Client patching guides (Windows, Android, iOS)
│   ├── windows/     # Windows client (2006-2021)
│   ├── android/     # Android client
│   └── ios/         # iOS client
├── rccservice/      # RCCService guides
│   └── windows/
├── studio/          # Roblox Studio
│   └── windows/
├── infrastructure/  # Network, API, Site infrastructure
│   ├── network/
│   ├── api/
│   └── site/
├── knowledge/       # Knowledge base, references, external links
├── utilities/       # Tools and utilities
└── misc/            # Miscellaneous
```

## Usage

Run with PHP's built-in server:
```bash
php -S localhost:8000 -t Wiki/
```
